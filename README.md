> description
------------------------------------------------------------
Yamifood is a modern and fully responsive restaurant website
built using only HTML5 and CSS3.

This project focuses on clean semantic structure, responsive
layout techniques, and professional UI styling without using
JavaScript, frameworks, or external libraries.

Goal:
Master frontend fundamentals through pure structure and styling.
------------------------------------------------------------
> features
------------------------------------------------------------
[+] Responsive layout
[+] Clean navigation bar
[+] Hero landing section
[+] About section
[+] Menu showcase
[+] Food gallery
[+] Contact form layout
[+] Footer section
[+] Mobile-friendly design
