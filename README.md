## ✨ About the Project

> A **fully responsive restaurant website** built with nothing but HTML5 and CSS3. Yamifood showcases a modern, elegant design suitable for restaurants, cafés, and food businesses — without touching a single framework or JavaScript library.

This project is a deep dive into **frontend fundamentals** — semantic markup, responsive layouts with Flexbox and CSS Grid, and professional UI polish achieved through pure craft.

---

## 🎯 Key Features

| | Feature | Description |
|---|---|---|
| 🍴 | **Multi-Page Layout** | Fully navigable site with 6 distinct pages |
| 📱 | **Fully Responsive** | Mobile-first design, flawless on all screen sizes |
| 🧭 | **Clean Navigation** | Smooth, intuitive navbar across all pages |
| 🍜 | **Menu Showcase** | Elegant food menu with structured layout |
| 🖼️ | **Food Gallery** | Visual gallery to display dishes beautifully |
| 📅 | **Reservation Page** | Dedicated table booking layout |
| 🪶 | **Zero Dependencies** | Pure HTML5 + CSS3 — no JS, no Bootstrap, no frameworks |
| ⚡ | **Instant Load** | No build step, no bundler — just open and run |

## 🚀 Getting Started

No installation. No build step. Just clone and open.

```bash
# Clone the repo
git clone https://github.com/Dev-Chandan404/Yamifood-Restaurant.git
cd Yamifood-Restaurant

# Open directly in your browser
open index.html

# Or serve locally
npx serve .
# → http://localhost:3000
```
